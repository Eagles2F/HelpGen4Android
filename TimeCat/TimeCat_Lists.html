Lists of tasks are results of searches in the task database. There are two standard lists: active and completed.
The active tasks are current if the deadline is in the future. They become expired when the current time goes past the deadline. Additionally to the standard lists, the user can define new search criteria resulting in custom lists. {ToDo: For application responsiveness purposes, the maximum number of tasks displayed in a list is limited in settings.} All the lists have a common interface described below.

Tasks are represented in a list by the name and an information field.
The information field lists:

  • the remaining time until the deadline for the active tasks;
  • the expiration time for the expired tasks;
  • the completion time for the completed tasks.

The order of the tasks in the active list is given by a score determined by the priority of the task and by the percentage of the time remaining to completion if the deadline is in the future. (check if true!!) {ToDo: Figure out the formula}

The order of the tasks in the completed list is given by completion time.

The order of the tasks in a custom list is given by the search criteria defined by that list.

The operations that can be performed from the list screen are listed below:

Add a new task: Touching the icon on the top bar creates a new task and opens the edit screen for it. The menu generated by a long touch on an existing task allows to generate a new task with a higher priority than the current one.

Edit an existing task: Touching the task in any list opens the edit screen.

Delete a task: Swiping right to left over a task shows a quick delete button. {ToDo: cancel the delete button} Deleted tasks cannot be recovered. Other ways to delete the task is using the menu generated by a long touch, or through the block operations.

Move a task: Swiping left to right over a task toggles on and off the "complete" status. Alternatively, the task can be marked as completed or active from the edit screen. Another way to move the task between lists is using the menu generated by a long touch.

Reorder tasks: Tasks can be reordered by changing the priority or the deadline in the <edit> screen. {ToDo: Tasks can be reordered by dragging them vertically in the list.}

Block operations: By using the icon, tasks can be selected to perform block operations like deletion, complete/uncomplete, select/unselect all, {ToDo export}. Touching the icon again cancels the block mode.

Statistics: By using the icon, statistics can be created from the tasks in the list.
