Lists are results of searches in the database of tasks as following:

  • the Active list contains all tasks in the Active or Expired state;
  • the Comppleted list contains all tasks in the Comppleted state;
  • a Custom list contains all tasks resulted from a custom search.

To create a new search, touch "New Search" under the "Custom Lists" menu. The last 3{-6 maybe option in settings} sets of search criteria are stored under the "Custom Lists" menu under the desired names. The searches not listed under "Custom Lists" are forgotten.

Each time an existing search is selected, a new search is conducted using the selected criteria.

A new search is based on a set of parameters explained below:

List Name is the name of the new search and of the list generated by it.
Mandatory.
Default: empty
Filter: {ToDo}

Order By specifies the parameter to be used for ordering the new list.
Possible values: Priority, State, Task Name
Default: Priority

Importance specifies the importance of the included tasks.
Possible values: Any, LOW, MODERATE, IMPORTANT, CRUCIAL
Default: Any

Tags specifies the list of tags included on the list.
Default: {all tags}

{ToDo: Deadline between ... and ... selects tasks based on deadlines.
Default = all deadlines}

{ToDo: Created between... and ... selects tasks based on the time when they were created.
Default = all times}
