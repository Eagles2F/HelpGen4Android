Statistics can be built based on the properties of the tasks {ToDo: from the current list}. By touching the {piechart} button in the list , the statistics is generated from all the tasks selected by the search criteria of the list.

Statistics presentation The results are listed numerically, as percentages, and as pie charts. Classification can be done by importance, by state, and by tag, as selected by the buttons at the bottom of the screen. If a task has multiple tags, it is counted once for each of the tags in the tag-based statistics. {ToDo: If the number of slices in the pie charts exceed the number of colors specified in settings , the first N-1 slices would correspond to the highest percentages, while slice N will contain the sum of the smaller percentages.}
