Tasks are the atoms of the TimeCat application. They are stored in a database, and can be used for searches or statistics . Task details can be visualized or edited from the task editor screen obtained by touching the task in a list or by creating a new task from one of the list screens . The icon saves the task and exits the editor. This section describes the task editor screen.

State: State of the task reflecting the list (active or completed) and the deadline (active or expired).
Default: expired.

  • Active means the task is in the active list and the deadline is still in the future.
  • Expired means the task is in the active list but the deadline is in the past.
  • Completed means the task is in the completed list.

Importance: Field that can rate the importance of the task.
Values can be: low, moderate, important, crucial.
Default: moderate

Deadline: The deadline of the task; used to decide the state of the task (active vs. expired) and to order the tasks in the active list.
Default: current time

Start: Start time of the task. Used for including tasks searches and for calculating the completion percentage.
Default: current time at task creation

Priority: The numeric priority that influences the order of tasks in the active list.
Range: 0-9999
Default - depending on importance:
